# 🔴 Omni-Man Frost

A **BetterDiscord theme** inspired by Omni-Man from *Invincible*.

Built with FrostedGlass, custom glass effects, red cinematic colors, and the Wood Block CG Regular display font for an aggressive comic-style Discord experience.

![Omni-Man Theme Preview](images/omni-man-wallpaper.png)

---

## ✨ Features

### 🎨 Visual Style
- Omni-Man inspired background
- Dark red / black color palette
- Frosted glass Discord UI
- Rounded windows
- Custom gradients
- Cinematic atmosphere

### 🔤 Custom Font
- Wood Block CG Regular font support
- Automatic font loading through GitHub
- Improved font spacing and readability

### 💬 Discord Improvements
- Larger chat text
- Larger usernames
- Bigger DM names
- Improved channel readability
- Cleaner text rendering

### 🟢 Extra Addons
Includes:

- FrostedGlass
- RadialStatus
- Windows Titlebar addon
