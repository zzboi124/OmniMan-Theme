# ⚡ Invincible-Verse Discord Themes

Three BetterDiscord themes, one for each side of the family. Not just a wallpaper and a font swap — **each theme has signature motion and interaction design pulled straight from that character's power set.** If someone's scrolling through your setup and asks "wait, what theme is that," this is the collection where they immediately know which one they want.

| Theme | Who it's for | The thing that sells it |
|---|---|---|
| 🔵 **Invincible** | "I want to feel like Mark" | Flight-trail hover glow, comic-panel impact cracks on pings, glasses-glint avatar shimmer |
| 🔴 **Omni-Man** | "I want the menace" | A red pulse that slowly escalates the longer you're online, blood-bleed message hovers, cape-heavy modal easing |
| 🩷 **Atom Eve** | "I want the elegance" | Messages *materialize* like matter assembling, telekinetic bloom on hover, a construct-ring around avatars |

---

## 🔵 Invincible — *"You're not gonna believe how far this suit lets you push a theme."*

Mark's power is flight and impact — so the theme moves like it.

- **Flight-trail hover** — hover a channel or DM and a blue/gold energy streak trails in from the edge and fades, like an afterimage
- **Impact-crack pings** — get mentioned and the highlight doesn't just glow, it *cracks in* with a comic-panel flash, then settles
- **Glasses-glint avatars** — a thin diagonal light sweep crosses every avatar every few seconds
- **Sonic-boom send** — hit send and get a quick expanding ring off the button, like breaking the sound barrier
- Deep royal blue → gold gradient palette matching the suit exactly

## 🔴 Omni-Man — *"This one's not messing around."*

Built around the show's central trick: the mask slipping a little more every time.

- **The Rage Escalation** — the red glow around your active channel bar slowly intensifies over a 90-second cycle, then eases back down. The longer you're active, the heavier it gets
- **Blood-bleed hover** — hover a message and a dark red bleed creeps in from the left edge before settling
- **Cape-flutter easing** — modals and popouts open with a heavier drag-then-snap curve instead of a flat fade
- **Impact-crater pings** — mentions land with a hard flash and a shockwave ring, not just a static highlight
- Dark red/black cinematic palette, frosted glass throughout

## 🩷 Atom Eve — *"Watch how this comes together."*

Eve manipulates matter — so the UI itself feels assembled, not just rendered.

- **Matter assembly** — new messages fade and scale in with a soft blur-to-focus materialize effect, like she's shaping them out of nothing
- **Telekinetic bloom** — hover a message and a soft pink/white construct-outline blooms outward from the edge
- **Construct ring** — hover an avatar and a thin dashed ring rotates slowly around it
- **Send burst** — sending a message scatters a tiny particle burst near the send button
- Soft pink → white gradient with a gentle shimmering scrollbar trail

## 🎨 Shared Foundation

All three themes share the same base so switching between them is seamless:

- **FrostedGlass** core with rounded windows and custom gradients
- **Wood Block CG Regular** display font, auto-loaded, with improved spacing/readability across chat, usernames, DMs, and channels
- **RadialStatus** + **Windows Titlebar** addons included
- Full `prefers-reduced-motion` support — every signature animation gracefully disables for users who need reduced motion, no exceptions

## 📥 Installation

**Requires [BetterDiscord](https://betterdiscord.app/) already installed.**

1. Download the `.theme.css` file for the character you want:
   - [`Invincible.theme.css`](./Invincible.theme.css)
   - [`OmniMan.theme.css`](./OmniMan.theme.css)
   - [`AtomEve.theme.css`](./AtomEve.theme.css)
2. Open Discord → **User Settings → BetterDiscord → Themes**
3. Click **Open Themes Folder**
4. Drop the downloaded file into that folder
5. Toggle the theme on in the Themes tab
6. Restart Discord (or just reload with `Ctrl+R`) if fonts/background don't load immediately

> 💡 Fonts and images auto-load from this repo's `fonts/` and `images/` folders via jsDelivr CDN — no manual asset placement needed.
